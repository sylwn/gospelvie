Verve Mobile automatically detects standard and advanced mobile devices and displays a compatible WordPress mobile theme.

== Description ==

The Verve Mobile plugin automatically detects the user's device and loads a compatible WordPress mobile theme. 

**Key Features:**

* Automatic detection of mobile devices
* Easily monetizes your mobile site with seamless integration into The Verve Network
* Advanced mobile statistics including unique visitors, visits, and more!
* Automatically formats content and resizes images
* Includes two pre-installed mobile themes

== Installation ==

1. Unzip and place the 'verve-mobile-plugin' folder in your 'wp-content/plugins' directory
2. Click the 'Activate' link for Verve Mobile on your Plugins page (in the WordPress administration page)

== Frequently Asked Questions ==

= I am getting an error that it was unable to install correctly. What do I do? =

Please try deactivating and reactivating the plugin. 


= Images are not showing for advanced mobile devices =

This is most likely due to folder permissions being incorrect.

Please set permissions for the 'cache' folder located inside of the 'plugins/a-verve-mobile' folder to 777.


= My site shows a blank page for a mobile device. What do I do? =

Check to make sure that your server is using PHP5. If it is not, please switch to PHP5.

If it is using PHP5, first try selecting a different mobile theme. If this does not work, there was a problem with activation.

Please try de-activating the plugin and re-activating it.
